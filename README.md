# Super Tontine

Onchain tontine.

## Install

```
pnpm install
```

Create a `.env` file:

```
cp .env.template .env
```

Add your own keys in the `.env` file. 

## Test 

```
pnpm test
```

## Deploy 

```
pnpm tontine
```

## Use

Mint USD:

```
pnpm mint 42
```

Send:

```
pnpm send 8
```
